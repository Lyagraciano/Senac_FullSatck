Linguagem WEB

5 Linguagens básicas

Se você é um iniciante na área ou deseja seguir carreira como desenvolvedor , precisa saber que o Desenvolvimento Web é dividido, basicamente, em dois grupos: desenvolvimento de front-end e desenvolvimento de back-end. Cada um deles possui suas próprias características e apresentam conjuntos distintos de ferramentas e tecnologias.Assim serão necessário utilizar as seguintes linguagens:

  • Front-end
    • HTML
    • CSS
    • JavaScript
  • Back-end
    • PHP
    • SQL
